To compile code simply type make in terminal on any PC that has make and c++11 installed

Sudoku simply reads in a .txt file formed as in the database folder.
To run Sudoku on files in the database folder run test2.sh

Sudoku2 takes in the number of positions p to fill in between 1 and 81 as input, generates a 
random puzzle of size 9x9 with p numbers filled in. It will then solve the puzzle and give the time taken to do so as an output.
